# API Documentation

Base URL: `http://localhost:3000` (development)

## Health Check

Check if the application is running.

```
GET /api/health
```

**Response** `200 OK`

```json
{
  "status": "ok",
  "timestamp": "2026-04-17T23:30:00.000Z"
}
```
